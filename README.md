# Python Web Scraper

## Description
This project scrapes product names and prices from a website
and exports the data into a CSV file.

## Features
- Extracts product titles
- Extracts product prices
- Saves results into CSV format

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## How to Run
1. Install required libraries:
   pip install requests beautifulsoup4 pandas

2. Run:
   python web_scraper.py
